Engineering Support

Math and formulas

Engimo comes with out-of-box support for math and formulas based on MathJax.

Diagrams and charts

Engimo extends Minimo theme with support of:

  • Graphviz diagrams
  • Digital waveforms diagrams
  • 2D Charts
  • Message Sequence Charts
  • Flowcharts

Deprecated pages

Support for marking pages as deprecated.

Online vs offline

Most of these features are available when working online, since some of the work performed relies on JavaScript resources available in the Internet.

feature online offline
maths and formulas ✓ -
graphviz ✓ ✓
waveforms ✓ ✓
deprecated pages ✓ ✓
